# Library Management System (OOP Python)
Library management System merupakan sebuah sistem manajemen perpustakaan sederhana berbasis Object-Oriented Programming (OOP). Pada sistem ini User dapat menambahkan buku atau majalah, menampilkan semua item, serta mencari item berdasarkan judul atau ID.

Pada Program ini diterapkan:
- Abstract Class
- inheritance
- Encapsulation
- Polymorphism
- Input dari User
